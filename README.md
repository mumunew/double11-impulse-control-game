# 双十一理性消费游戏

这是一个帮助用户识别并抵制冲动消费念头，培养理性消费习惯的游戏。

## 在线体验
- 🎮 [在线游戏地址](https://double11-impulse-control-game-92hviqxsp-mumunews-projects.vercel.app)
- 📦 [GitHub仓库](https://github.com/mumunew/double11-impulse-control-game)

## 游戏规则

- 玩家需要用发射器射击粉色的冲动消费念头（小恶魔图标，+20分）
- 放过青色的理性消费念头（小天使图标，+10分）
- 反之则扣分：击中理性消费-20分（第二关后-40分），放过冲动消费-10分
- 收集金币可以获得额外分数（+10或20分）

## 新增功能（V2.0）

### 1. 关卡升级弹窗
- 完成当前关卡后会显示精美的升级弹窗
- 弹窗显示精美升级弹窗，包含新关卡信息、新目标分数和鼓励信息
- 玩家需要点击"继续挑战"按钮才能进入下一关

### 2. 射掉理性消费加倍扣分
- 从第二关开始，射掉理性消费的扣分从-20分增加到-40分
- 如果理性消费带有促销字样，扣分进一步加倍到-80分

### 3. 关卡速度增加
- 每通过一关，消费念头的下落速度增加
- 玩家移动速度也会随着关卡提升，保持游戏平衡
- 关卡间隔时间减少，增加游戏难度

### 4. 第三关后出现促销字样
- 从第三关开始，40%的消费念头会带有"促销"字样
- 带有促销字样的冲动消费得分加倍（40分）
- 带有促销字样的理性消费扣分加倍（80分）
- 促销消费有特殊视觉效果：金色边框、闪烁效果和"×2"标记

## 操作方式

- 鼠标点击：发射子弹
- 鼠标移动：控制发射器左右移动
- 键盘方向键：控制发射器左右移动
- 空格键：发射子弹
- P键：暂停游戏

## 游戏特色

- 精美的精美的视觉效果和动画
- 多种控制方式，支持鼠标、键盘和触摸操作
- 关卡系统，难度逐渐增加
- 即时反馈和音效
- 响应式设计，适配不同设备
- 促销消费系统，增加游戏策略性

## 如何运行

1. 克隆或下载本项目
2. 打开index.html文件即可开始游戏

## 游戏机制

- 游戏采用关卡系统，每个关卡关卡需要完成20个消费念头（随关卡关卡增加）
- 每完成一个关卡，难度会增加（消费念头移动移动速度变快，出现频率增加）
- 达到目标分数可以晋级到下一关
- 游戏没有时间限制，可以可以无限游戏直到主动结束
